# Modify Array After Last Odd Element

Цей консольний застосунок на мові C# створює одновимірний масив цілих чисел, знаходить останній непарний елемент, і після нього вставляє кілька нових елементів зі значенням `lastOddIndex + 1`.

## 🔧 Функціонал

- Запитує розмір масиву у користувача.
- Генерує випадкові значення в діапазоні `[-n, n]`.
- Визначає індекс останнього непарного елемента.
- Додає **від 1 до 10** елементів після останнього непарного, заповнюючи їх однаковими значеннями.
- Виводить початковий і змінений масив у консоль.
